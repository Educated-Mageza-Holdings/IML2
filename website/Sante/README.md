## Customization

### Colors
Main colors are defined in `:root` in `styles.css`:
```css
--primary-green: #007A3D;
--primary-gold: #FFB81C;
--primary-red: #DE3831;
```

### Products
To modify products, edit the `.product-card` sections in `index.html`. Each product includes:
- Product icon (emoji)
- Gradient background color
- Product name and description
- Rating
- Feature badges

### Contact Information
Update contact details in the Contact section of `index.html`:
- Address
- Email addresses
- Phone numbers
- Social media links
